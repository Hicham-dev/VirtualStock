#How to deploy for developpers 

1 - install <a href="https://getcomposer.org/">composer</a>
2 - Execute the next commandes 

'''sh
cd ./VirtualStock
composer install
php artisan key:generate
'''
3 - create a vhost in your local server for the domaine 

'''sh
virtualstock.ma
'''
